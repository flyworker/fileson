Fileson module¶

Fileson class to manipulate Fileson databases.

class fileson.Fileson(runs: list = None, root: dict = {}, checksum: str = None) → None[source]¶

File database with versioning support in JSON format.

Normally only checksum is used, the other two (optional) parameters are mainly for load() and load_or_scan() methods.

Parameters:
  • runs (list) – A list of previous runs
  • root (dict) – Contents of previous run
  • checksum (str) – Checksum name or None

scan(directory: str, **kwargs) → None[source]¶

Scan a directory for objects or changes.

Every invocation creates a new ‘run’, a version to Fileson database. Only changes need to be stored. You can then use for example genItems() and pick only objects that were changed on a given run.

Parameters:
  • directory (str) – Directory to scan
  • **kwargs – Booleans ‘verbose’ and ‘strict’ control behaviour
